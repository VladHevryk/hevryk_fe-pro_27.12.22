• git status - статус изменения файлов
  • git add - добавить все измененные файлы
  • git commit - сохранение
  • git commit -m - сохранить сразу с описанием комита
  • git push - отправить данные на репозиторий (origin HEAD) - указание пути для сохранения
  • git branch - узнать на какой ветке я работаю
  • git checkout (название ветки) - перейти на другую ветку
  • git checkout -b (название ветки) - создать новую ветку
  • git commit -a - сохранить измененные файлы, если не добавляли новые
  • git log - история всех изменений и тд
